Document Classifier – File Description
main.py – Entry point of the application; takes file path input and prints JSON output.
txt_extractor.py – Detects file type and extracts readable text from TXT, PDF, and DOCX files.
classifier.py – Classifies the extracted text into predefined document categories using keywords.
BODDU-PRASANNA-Resume-AE.pdf – Sample PDF file used for testing resume classification.
EmptyWordDoc.docx – Sample DOCX file used to test invalid document handling.
